MY WORK

Project 01

The ticket booking system is a Java-based application designed to manage airplane seat reservations efficiently. It allows users to view seat availability, select preferred seats, and confirm bookings, providing a streamlined experience for both customers and airlines by automating the reservation process and reducing booking conflicts.

Project 02

The mark entering system is a Python-based application designed to streamline the process of inputting student scores. It allows educators to enter, update, and store marks efficiently, automatically calculating averages and generating performance summaries, making it easier to manage and analyze academic records for both students and teachers.

Project 03

The "20 x 2" board game is an interactive game created in the Programming I module, where a human player competes against a computer. Designed for a dynamic, turn-based experience, the game challenges players to make strategic moves, enhancing logical thinking and providing engaging gameplay with real-time feedback.

Project 04

The "Marinelife" web page, created using JavaScript, HTML, and CSS, showcases an engaging and informative look at marine ecosystems. Designed for the Web Design and Development module, it offers interactive elements, vibrant visuals, and easy navigation, providing users with an immersive experience to explore diverse marine species and habitats.
